## 概述

v0.3.5 标志着 jianwu 内核的 SaaS 化就绪。所有核心包已完成 storage 接口抽象和注入化改造，为独立的 SaaS 层（mouqin）铺平道路。

## 核心变更

- **Storage 接口全线迁移**：所有 8 个核心包均已依赖 `storage.Storage` 接口而非直接 OS 操作
- **MemStorage 完备**：16 个测试覆盖全部接口方法
- **Provider 装配重构**：消除全局可变变量，显式参数注入
- **Schema 迁移移除**：所有 workspace schema 迁移代码清理完毕

## 完整管线

```
grill → outline → scaffolding → expand [→ factcheck → revise →] review → finalize → export
```

支持 streaming 输出、corpus sync、embedding 索引缓存、章节增删改批操作。

## 下一步

SaaS 包装层 `mouqin` 在独立仓库中启动，提供 Web UI 和 API。
